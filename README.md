# Aleup Project: https://www.aleup.fi/ (back-end)

## Features:
- Query to database get the products with many complex input parameters: tag (category), name (product's name), market (market sell the product), discount,...
- There are also other features to get products like: get all tags, get products popular to introduce in hompage, get product min price and max price, get all markets, get markets by tag, get tags by market

## Technologies:
Nodejs, Appollo Server, Express, Graphql, Mongodb
